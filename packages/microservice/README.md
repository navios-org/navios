# @navios/microservice

Microservice framework using message-based communication for Navios.

## Installation

```bash
npm install --save @navios/microservice @navios/core @navios/queues @navios/di zod
```

or

```bash
yarn add @navios/microservice @navios/core @navios/queues @navios/di zod
```
